SmartYnvest

A portfolio management tool is a software application that helps investors to track their investments and make informed investment decisions. It can be used to track a wide range of investment assets, including stocks, bonds, mutual funds, and real estate. This portfolio management tool is developed in Node.js and React.js, all in TypeScript. It is a full-featured tool that includes all the essential features of a modern portfolio management tool.

Library Management System

A library management system is a software application that helps libraries to manage their collections, users, and operations. It can be used to automate tasks such as book cataloging, circulation, and reporting. This project was developed in Java Spring Boot & MySQL for the backend and React TypeScript for the frontend. I use this personal project to catalog and track my own books, whether I let them to someone, or whether I have read them, or which ones I would like to add to the collection.

Check it out here!

Banking System

This Banking System project is a full-stack web application developed using Java Spring Boot and React TypeScript. It provides a secure and user-friendly platform for customers to manage their finances effectively, make transfers, receive money, open and close savings accounts, etc. I developed this project to gain experience in building a banking system and learn about the challenges and opportunities that arise from such a project.
